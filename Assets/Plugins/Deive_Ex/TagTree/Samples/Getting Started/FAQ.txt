Q: Where are the Sample Tag File located?

A: By Default Tags are loaded from the Resources folder. Inside the Samples folder you'll find a folder called "Resources",
and inside it there's another folder called "Tags". You'll find the Tag File inside of it.

---

Q: My GameObjects Sample Scene has pink spheres! What do I do?

A: This is because the sample scene was created in the Built-in Render Pipeline, and you're probably using a different
Render Pipeline. To fix this, simply go to the "Materials" folder inside the "Getting Started" folder, select the materials 
and change the shader to some other compatible shader, like "Standard" or "Unlit".

---

Q: My GameObjects Sample Scene have a bunch of completely white buttons! There's no text inside them!

A: The texts inside this scene was created using Text Mesh Pro. You need to first import the Package from the Package Manager
(if you don't have it already) and then Essentials Package. Usually TMP prompts you to import the Essentials Package when
you select any TMP Text, but if it didn't, you can still import it by selecting "Window > TextMeshPro > Import TMP Essentials 
Resources" in the top bar.
