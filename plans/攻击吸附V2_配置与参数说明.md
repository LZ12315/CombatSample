# Magnetism — 配置说明（正式版）

> 说明见 **[`MagnetismV2_收束说明.md`](MagnetismV2_收束说明.md)**（文件名保留 V2 历史，内容已更新）。

## Timeline

1. 使用 **Action Magnetism** 轨道，在需要锁向的时段放置 Clip。
2. Clip 上 **Use Combat Target** 使用当前 `CombatTarget`；否则指定 **Custom Target**。

## MagnetismConfig（仅旋转 + 门控）

| 字段 | 说明 |
|------|------|
| **Max Distance** | 玩家根与目标 pivot **水平距离**超过此值时，本 Clip **不旋转**（0=不限）。 |
| **Rotate To Target** | 是否朝向目标。 |
| **Rotation Mode** | None / Instant Snap / Angular Speed。 |
| **Rotation Angular Speed** | 度/秒；Mode 为 Angular Speed 时有效。 |
| **Rotation Axis** | 一般用 Yaw Only。 |
| **Debug Log** | 门控跳过时的日志。 |

**不再包含**：根位移、胶囊间隙、`idealSurfaceGap` 等；站位与推挤见 **`CombatSample.PhysicsInteraction`**。
