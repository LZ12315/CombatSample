# 攻击吸附 V2 — 怎么配、参数什么意思

面向：**Timeline + 招式动作**。吸附逻辑是：**玩家 Actor 根节点** 相对 **敌人身上的 CapsuleCollider 外壳** 保持一段「合适距离」，远了拉近、近了推开，并可选转向敌人。

---

## 1. Unity 里怎么挂

1. 打开招式的 **Timeline**（Playable Director）。
2. **右键 → Add → Action Magnetism V2**（或项目里显示的绿色 Magnetism V2 轨道类名：`ActionMagnetismV2Track`）。
3. 轨道绑定方式需与你们项目里 **其他 Action 轨道一致**（通常和 Animation / HitBox 一样绑 **玩家** 或带 `Actor` 的对象），这样 Clip 播放时才能拿到 `Actor`。
4. 在需要吸附的 **时间范围** 上放 **ActionMagnetismV2Clip**，拖长/缩短 Clip 即吸附持续时长。

---

## 2. Clip 上两项：目标（Target）

| 项 | 说明 |
|----|------|
| **Use Combat Target**（默认勾选） | 目标 = `Actor` 上战斗组件里的 **当前战斗目标**（`CombatTarget`）。一般打怪就用这个。 |
| **Custom Target** | 仅当 **取消勾选** Use Combat Target 时有效：手动拖一个 **Transform** 当作吸附参考（敌人根、某标记点等）。 |

**敌人侧前提**：目标物体上（自身或子物体）要有 **`CapsuleCollider`**。没有的话吸附**不会位移**（可开 `debugLog` 看日志）。

---

## 3. MagnetismConfig 参数（核心）

在 Clip 里展开 **Config**。

### 3.1 Distance

| 参数 | 含义 | 怎么调 |
|------|------|--------|
| **Max Distance** | 玩家根与敌人 **Transform  pivot** 的 **水平距离** 超过此值时，**整段 Clip 不吸附**（不位移、旋转仍可按下面配置执行，但通常一起用）。 | **0** = 不限制。想「太远就不吸」：设为略大于你希望触发吸附的最大距离，例如 `2`～`4`（米，按角色比例）。 |

### 3.2 Surface gap（最重要）

系统算的是：**Actor 根的世界坐标** 到 **敌人胶囊壳** 的 **最短距离**（间隙，米）。

| 参数 | 含义 | 怎么调 |
|------|------|--------|
| **Ideal Surface Gap** | 希望保持的 **目标间隙**（米）。 | 穿模就 **略增大**；站得太远打不到就 **略减小**。常见可从 **0.25～0.5** 试。 |
| **Gap Dead Zone** | 实际间隙在 **`Ideal ± DeadZone`** 里时 **不再推拉**，避免哆嗦。 | 太小会抖，太大则「acceptable 区间」过宽。可从 **0.03～0.1** 试。 |

直观理解：

- 间隙 **大于** `Ideal + DeadZone` → **拉近**（Pull）。
- 间隙 **小于** `Ideal - DeadZone` → **推开**（Push）。

### 3.3 Approach（位移方式与速度）

| 参数 | 含义 | 怎么调 |
|------|------|--------|
| **Approach Mode** | **Speed Move**：每帧按速度往区间里修。**Instant Move**：**第一帧**一次性把水平方向上的间隙修到理想（之后本 Clip 内不再因间隙做连续位移）。 | 想要「蓄力开始瞬间卡好距离」可试 Instant；想要顺滑贴近用 Speed。 |
| **Pull Speed** | 间隙 **偏大** 时，根节点 **靠近敌人** 的速度（米/秒）。 | 与招式时长匹配：Clip 很短可略大；太长可略小避免「冲过头」。 |
| **Push Speed** | 间隙 **偏小** 时，沿 **离开表面** 方向推的速度（米/秒）。 | 常与 Pull 设成接近；若总穿模可略 **提高 Push**。 |

### 3.4 Rotation

| 参数 | 含义 | 怎么调 |
|------|------|--------|
| **Rotate To Target** | 是否让角色 **朝向敌人**（水平）。 | 需要刀口对着人则勾选。 |
| **Rotation Mode** | **None** 不转；**Instant Snap** 每帧（逻辑上）瞬转；**Angular Speed** 用角速度平滑转。 | 一般动作游戏用 **Angular Speed**。 |
| **Rotation Angular Speed** | 度/秒。Mode 为 Angular Speed 时有效；**0** 等价于不转（与代码一致）。 | 如 `360`～`720`。 |
| **Rotation Axis** | **Yaw Only** 只水平转（常用）；**Full** 全轴（少用）。 | 默认 Yaw Only。 |

### 3.5 Debug

| 参数 | 含义 |
|------|------|
| **Debug Log** | 勾选后会在 Console 打 **为何跳过吸附**（例如超 Max Distance、缺 Capsule）等，调参数时打开，调完关掉。 |

---

## 4. 推荐调参顺序

1. 确认敌人有 **CapsuleCollider**，**Max Distance** 先设 **0** 排除门控。  
2. 调 **Ideal Surface Gap** + **Gap Dead Zone**，直到「不穿模、距离顺眼」。  
3. 再调 **Pull / Push Speed**（或换 **Instant** 看首帧手感）。  
4. 最后加回 **Max Distance** 限制远程不吸。  
5. 微调 **旋转** 与 Clip **起止时间** 对齐蓄力/前摇。

---

## 5. 和旧轨道区别（别混用）

- 请用 **Action Magnetism V2** 轨道，配置在 Clip 的 **MagnetismConfig** 里。  
- 旧的 **Action Magnetism**（紫色那条）已 **Obsolete**，参数完全不同（`magnetSpeed` 等），新招式不要用。
