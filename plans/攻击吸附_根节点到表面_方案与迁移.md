# 攻击吸附 — 根节点 ↔ 敌人碰撞体表面（双向区间）方案与迁移

> **⚠️ 工程已再次收束**：Magnetism V2 **不再做根位移**，仅 Clip 内旋转；根表面间隙方案已从代码移除。请以 **[`MagnetismV2_收束说明.md`](MagnetismV2_收束说明.md)** 为准。下文保留作历史记录。

> **配置与参数怎么调（面向使用）**：见 **[`攻击吸附V2_配置与参数说明.md`](攻击吸附V2_配置与参数说明.md)**。

> **定位**：取代「根贴 pivot + 武器锚点表面修正」的混用逻辑。  
> **原因**：吸附发生在**命中前/蓄力段**，武器中点世界位置**不代表**命中帧几何，故改用**玩家根节点**相对**敌人受击胶囊表面**的距离做**双向**修正（远了拉近、近了推开），参数靠试错区间收敛。  
> **关系**：本文档为**新权威**；`攻击吸附与战斗锚点_最终实现方案.md` 中 **V2 武器锚点 + 武器表面** 相关章节**作废**，实现落地后以本文为准。

## 1. 行为规格（策划语义）

### 1.1 生效条件

- 与现有一致：仅在 Timeline **Magnetism Clip 播放时段**内执行。
- **`maxDistance`（保留）**：玩家根与敌人参考点（见下）的**水平距离**大于此值时，**整段不吸附**（0 = 不限制）。  
  - *可选细化*：也可定义为「根到胶囊表面间隙」已超过某外圈时再禁用——实现阶段二选一，文档先以**水平离敌人 Transform** 为外圈 gate（改动小）。

### 1.2 核心标量：根 ↔ 表面「间隙」

- **采样点**：`actor.transform.position`（或约定只用 **XZ**，Y 与敌人胶囊计算时统一用 actor 的 Y 做水平投影点，避免身高抖几何——实现时固定一种并写进代码注释）。
- **敌人几何**：`CombatTarget` 上 `CapsuleCollider`（`GetComponent` → `GetComponentInChildren`），与现 Session 一致。
- **间隙 `gap`**：采样点到胶囊壳的**最短距离**（米）。在壳外为正；在壳内为负（或 0 + 穿透标志——实现统一一种）。

### 1.3 双向修正（区间内不动）

- 给定**目标间隙** `idealSurfaceGap`（米）与**死区** `gapDeadZone`（米）：  
  - 允许区间 **`[ideal - dead, ideal + dead]`** 内：**不做位移修正**（或只做旋转）。  
- **偏远**（`gap > ideal + dead`）：沿**靠近敌人**方向移动根，**限速** `pullSpeed`（米/秒）。  
- **偏近**（`gap < ideal - dead`）：沿**离开敌人表面法线**（或水平投影后的外法线）移动根，**限速** `pushSpeed`（米/秒）。  

> 与旧逻辑对比：旧版「`attachDistance` 内不再推」是**单向停**；新版是**过近要主动推离**。

### 1.4 旋转

- **保留**现有 `MagnetismConfig` 旋转块：`rotateToTarget`、`rotationMode`、`rotationAngularSpeed`、`rotationAxis`。  
- 朝向仍建议 **水平面指向敌人**（`dir` 与现有一致）。

### 1.5 瞬移模式（可选保留）

- **`InstantMove`**（若保留）：可在**首帧**将根**一次**拍到区间内最近边界/中点；之后帧仅用 **Speed** 双向微调。  
- 若简化为**仅连续吸附**：可**删除** `InstantMove` 枚举值（破坏性较大，迁移时可用 `[Obsolete]` 标在旧值上）。

## 2. `MagnetismConfig` — 字段变更表

### 2.1 删除（不再用于 V2 吸附）

| 字段 | 原用途 |
|------|--------|
| `attachDistance` | 根相对敌人 pivot 的「方案一贴身」 |
| `weaponAnchorId` | 从 Actor 表取武器点 |
| `useWeaponSurfaceClearance` | 武器点单向表面修正 |
| `surfaceClearance` | 武器点目标间隙 |
| `maxRootCorrectionPerFrame` | 武器修正每帧上限（可被 `pushSpeed/pullSpeed` 替代） |

### 2.2 保留

| 字段 | 说明 |
|------|------|
| `maxDistance` | 超过则整段不吸附（语义见 §1.1，若改 gate 算法再更新 Tooltip） |
| `rotateToTarget` / `rotationMode` / `rotationAngularSpeed` / `rotationAxis` | 旋转 |
| `debugLog` | 调试 |

### 2.3 新增（建议命名，实现时可微调）

| 字段 | 类型 | 说明 |
|------|------|------|
| `idealSurfaceGap` | float | 根采样点 ↔ 胶囊表面的**目标间隙**（米），试错主旋钮 |
| `gapDeadZone` | float | 理想值±死区内不位移 |
| `pullSpeed` | float | 偏远时靠近速度（米/秒） |
| `pushSpeed` | float | 偏近时推开速度（米/秒） |

### 2.4 `approachMode` / `approachSpeed`

- **方案 A（推荐）**：删除 `MagnetismApproachMode` 与 `approachSpeed`，位移**仅**由 `pullSpeed`/`pushSpeed` 表达。  
- **方案 B**：保留 `approachSpeed` 作为 `pullSpeed`/`pushSpeed` 未设时的默认值，减少序列化破坏——迁移友好。

## 3. `MagnetismCapsuleGeometry` — 修改

### 3.1 重命名/泛化 API（建议）

将「武器中心」参数改为**任意世界点**：

```csharp
// 原 TryGetClearanceInfo(weaponCenterWorld, ...) 
// 改为：
public static bool TryGetClearanceInfo(
    Vector3 samplePointWorld,
    CapsuleCollider enemyCapsule,
    out Vector3 closestSurfaceWorld,
    out Vector3 outwardNormal,  // 从壳指向采样点外侧（推远采样点时根沿 +n 移）
    out float signedGap);
```

- **`ComputeRootDeltaForClearance`（基于武器单向缺口的旧版）**：**删除**或改为内部仅被新函数调用。  
- **新增**（框架签名）：

```csharp
/// <summary>
/// 单帧根位移：使 gap 向 [ideal-dead, ideal+dead] 靠拢。返回世界空间位移（水平投影由 horizontalOnly 控制）。
/// </summary>
public static Vector3 ComputeRootDeltaForGapBand(
    Vector3 rootWorld,
    CapsuleCollider capsule,
    float currentGap,
    Vector3 outwardNormal,     // 与 TryGetClearanceInfo 一致
    float idealGap,
    float deadZone,
    float pullSpeed,
    float pushSpeed,
    float deltaTime,
    bool horizontalOnly,
    float maxStepMagnitude);     // 0=不限制，或每帧硬帽
```

实现要点：`currentGap` 与 `idealGap` 比较差值，决定沿 `±outwardNormal`（或水平化后的切向+法向组合）**拉/推**；步长 `min(|error|, speed * dt)`。

*注：`currentGap` 也可在方法内用 `rootWorld`+`capsule` 重算，签名可合并。*

## 4. `ActionMagnetismSession` — 重写要点

### 4.1 构造函数

```csharp
public ActionMagnetismSession(Actor actor, Transform combatTarget, MagnetismConfig config)
// 删除参数：Transform weaponCenterTransform
```

### 4.2 `Tick` 伪代码

```
if target/actor/config/capsule invalid -> return (debugLog)
if maxDistance gate failed -> return

root = actor.transform.position
gap, normal, closest = TryGetClearanceInfo(rootSample, capsule)  // rootSample 定义见 §1.2

delta = ComputeRootDeltaForGapBand(..., gap, normal, ideal, dead, pull, push, dt, horizontalOnly, maxStep)
ApplyMoveHorizontal(delta)  // 与现 ApplyMoveTowards 类似：根位移，CC 关开

旋转：同现有 dir + ActorMovement
```

### 4.3 删除的逻辑块

- 整段 **`attachDistance` + `desiredPos`**。  
- 整段 **`useWeaponSurfaceClearance` + 武器 Transform**。  
- **`InstantMove` 分支**：若产品仍要首帧拍位，单独小函数；否则删除 `_didInstantMove` 与分支。
